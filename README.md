# Shooting Game (Windows)

A simple 2D shooting game built with **Python** and **pygame**.  
This game is distributed as a **Windows executable**, so you can play it without installing Python or any additional libraries.

---

## How to Play

### System Requirements
- Windows 10 / Windows 11
- Keyboard
- (Optional) Speakers or headphones for sound

No other software is required.

---

### Download & Run

1. Go to the **Releases** page of this repository and download the ZIP file from **Assets**
2. Extract the ZIP file
3. Double-click `shooting.exe` to start the game

> Note: On first launch, Windows may show a security warning.  
> If you see “Windows protected your PC”, click **More info → Run anyway**.

---

## Controls

- Arrow Keys : Move your ship
- Space : Shoot
- Shift : Slow movement (precision control)
- R : Restart (after Game Over)
- N : Next stage (after clearing a stage)
- ESC : Quit the game

---

## Game Features

- Multiple enemy types with different movement patterns
- Boss battles with increasing difficulty
- Power-ups:
  - Rapid fire
  - Spread shot
  - Piercing shot
  - HP recovery
- Stage-based progression
- Simple visual effects and sound effects
