# Shooting Game

Python と pygame で作成した、縦スクロール型シューティングゲームです。  
キーボード操作で自機を動かし、敵やボスを撃破してステージを進めます。

---

## 🎮 ゲーム概要

- ジャンル：縦スクロール・シューティング
- プレイ人数：1人
- 対応OS：Windows（Python 実行環境）

---

## 🛠 必要なもの（最初に1回だけ）

### 1. Python（必須）
- Python **3.10 以上**
- https://www.python.org/

※ インストール時に  
**「Add Python to PATH」** にチェックを入れてください。

### 2. pygame（必須）
Python をインストール後、PowerShell で次を実行します：

```bash
pip install pygame

▶ 起動方法

このリポジトリを ZIP でダウンロード

ZIP を展開

展開したフォルダで PowerShell を開く

次を実行：

python shooting.py


ゲームウィンドウが起動します。

🎮 操作方法
操作	キー
移動	矢印キー
ショット	SPACE
低速移動	SHIFT
リスタート	R
終了	ESC
